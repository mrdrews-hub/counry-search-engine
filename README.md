## Run Project Locally

Clone the project

```bash
  git clone https://github.com/mrdrews-hub/counry-search-engine.git
```

Go to the project directory

```bash
  cd counry-search-engine
```

Install dependencies

```bash
  npm install
```

Start

```bash
  npm run dev
```


## Tech Stack

**Client:** VueJS, daisyUI
